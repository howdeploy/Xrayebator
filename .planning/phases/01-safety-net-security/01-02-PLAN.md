**SEC-04: Fix install.sh key extraction for both xray x25519 output formats**


Replace lines 124-125 with grep patterns that support BOTH formats:
```bash
KEYS_OUTPUT=$(/usr/local/bin/xray x25519 2>&1)
# Support both old format (Private key:/Public key:) and new format (PrivateKey:/Password:)
PRIVATE_KEY=$(echo "$KEYS_OUTPUT" | grep -E "^Private" | awk '{print $NF}')
PUBLIC_KEY=$(echo "$KEYS_OUTPUT" | grep -E "^Public|^Password" | awk '{print $NF}')
```

`grep -E "^Private"` matches both `Private key:` and `PrivateKey:`. `grep -E "^Public|^Password"` matches both `Public key:` and `Password:`. `awk '{print $NF}'` takes the last field regardless of separator format.
