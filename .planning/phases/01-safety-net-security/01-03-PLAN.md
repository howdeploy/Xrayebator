**1. migrate_routing_config() (lines 305-322 area)**

Replace the raw jq block:
```bash

    jq '.routing.rules += [{
      "type": "field",
      "network": "tcp,udp",
      "outboundTag": "direct"
    }]' "$CONFIG_FILE" > "$temp_config"

    if [[ $? -eq 0 ]] && [[ -s "$temp_config" ]]; then
      mv "$temp_config" "$CONFIG_FILE"
```

